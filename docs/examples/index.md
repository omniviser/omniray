# Examples

Runnable examples showing common omniray and omniwrap usage patterns.

Each example is a standalone script you can copy and run directly:

```bash
python examples/<example_name>.py
```

| Example | What it shows |
|---------|--------------|
| [Basic Logging](basic-logging.md) | Custom wrapper that logs every function call with args and return values |
| [Execution Timing](basic-timing.md) | Measure function execution time with separate sync/async wrappers |
